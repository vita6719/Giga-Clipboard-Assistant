# Giga-Clipboard-Assistant
Умный ИИ-помощник на базе GigaChat. Разбирает ошибки в коде через буфер обмена и дает советы на русском языке.
# 🛡️ Giga-Clipboard Assistant
**Твой личный ИИ-наставник для работы с Python**

Этот проект помогает новичкам быстро разбираться с ошибками в коде, не перегружая компьютер сложными программами. 🧠✨

### ❓ Что делает этот код?
Программа работает как мостик между твоим редактором кода и нейросетью **GigaChat** от Сбера. 

**Всё очень просто:**
1. Ты видишь ошибку в консоли — **выделяешь её и нажимаешь Ctrl+C**. 📋
2. Запускаешь этот скрипт.
3. Программа сама забирает текст из буфера обмена, делает скриншот (на всякий случай) и отправляет запрос в облако Сбера. ☁️
4. Через пару секунд ты получаешь **понятный совет на русском языке**: что именно сломалось и как это исправить. 🛠️

### 🚀 Почему это круто?
- **Не тормозит систему**: Вся магия происходит на мощных серверах Сбера, твой ноутбук отдыхает. 💻☕
- **Умные подсказки**: GigaChat отлично понимает код и объясняет ошибки как живой учитель.
- **Минимум действий**: Не нужно открывать браузер и гуглить — помощь приходит прямо в окно программы.

### 🛠️ Как запустить:
1. Установи библиотеки: `pip install gigachat pyautogui pyperclip`.
2. Вставь свой ключ (Authorization Key) из кабинета Sber Studio в код.
3. Копируй любую ошибку и запускай файл!
